### Data Structure: 
+ Stack
### Storage Method: 
+ Singly Linked Node
### Key Methods:
+ Push()  
	+ Complexity: O(1)
+ Pop()
	+ Complexity: O(1)
+ IsEmpty()
	+ Complexity: O(1)
+ Top()
	+ Complexity: O(1)

### Description: 
+ It is a data structure container that 
removes items based on the LIFO 
(Last-In-First-Out) principle (also known
as FILO (First-In-Last-Out) principle.
